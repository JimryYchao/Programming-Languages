# C语言格式化输出
## 3. 长度修饰符(length)



### 精度
- `.数字`：小数点后位数（浮点数）或最小位数（整数）
- `.`：精度为0
- `.*`：从参数获取精度

## 6. 示例代码

```c
#include <stdio.h>

int main() {
    // 字符串格式化
    printf("%10s\n", "Hello");   // 右对齐，宽度10
    printf("%-10s\n", "Hello");  // 左对齐，宽度10
    
    // 整数格式化
    printf("%+d\n", 42);         // 显示+号
    printf("%#x\n", 42);         // 十六进制带0x前缀
    printf("%05d\n", 42);        // 前导零填充
    
    // 浮点数格式化
    printf("%.2f\n", 3.14159);   // 保留两位小数
    printf("%e\n", 3.14159);     // 科学计数法
    printf("%g\n", 3.14000);     // 自动省略尾随零
    
    // 动态宽度和精度
    printf("%*.*f\n", 8, 2, 3.14159);  // 宽度8，精度2
    
    return 0;
}
```
