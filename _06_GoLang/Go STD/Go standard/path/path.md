包 `path` 实现用于操作斜杠分隔路径的实用程序例程。`path` 包应该仅用于由正斜杠分隔的路径，例如 URL 中的路径。这个包不处理带有驱动器号或反斜杠的 Windows 路径；要操作操作系统路径，请使用 `path/filepath` 包。
